#2. Docker操作
###查看Docker版本信息
docker version

```[root@localhost ~]# docker version```
###启动Docker
```[root@localhost ~]# service docker start```
###查看Docker日志
```[root@localhost ~]# cat /var/log/docker```
###停止Docker
```[root@localhost ~]# service docker stop```
